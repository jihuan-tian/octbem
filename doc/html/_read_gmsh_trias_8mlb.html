Detailed Description

ReadGmshTrias - Read the triangle cells from Gmsh mesh file.

Parameters
gmsh_msh_file_nameThe file name of the Gmsh msh file.
gmsh_mesh_objThe struct object containg Gmsh mesh data, which includes the following fields: mesh_nodes: A list of nodes stored in N*M matrix, with N as the number of nodes, M as the space dimension. mesh_cells: A list of triangle cells stored in N*M matrix, with N as the number of cells, M as the number of nodes in a cell. number_of_nodes: Total number of nodes. space_dim: The space dimension used to define the node coordinates. number_of_cells: Total number of cells in the mesh. number_of_nodes_in_cells: Total number of nodes in each cell, which determines the polynomial order of the shape functions for describing the cell geometry. cell_geom_orders: The polynomial orders of the shape functions for describing the geometry of all cells. corner_node_local_indices_in_cells: Local indices of the corner nodes in all cells, where it is assumed that all the nodes are in the forward zigzag order. cell_ranges: The ranges of all cells. min_cell_range: The minimum mesh size \(h\). cell_surface_metrics: Calculate the surface metric from the reference cell to each real cell, which is the squared sum of the three two-coordinate-component Jacobians. cell_normal_vectors: Calculate the surface normal vector for each cell.
