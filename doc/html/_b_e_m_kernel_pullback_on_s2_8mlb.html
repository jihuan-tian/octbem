Detailed Description

BEMKernelPullback - Pullback a BEM kernel to the reference cell.

Parameters
xiA list of area coordinates on the reference cell \(\hat{K}_x\).
etaA list of area coordinates on the reference cell \(\hat{K}_y\).
nx_functorThe functor depending on area coordinates for calculating the cell normal vectors for \(K_x\).
ny_functorThe functor depending on area coordinates for calculating the cell normal vectors for \(K_x\).
kx_shape_functions_for_geometryA list of shape functions on the reference cell \(\hat{K}_x\) for describing the real cell geometry.
ky_shape_functions_for_geometryA list of shape functions on the reference cell \(\hat{K}_y\) for describing the real cell geometry.
kx_cell_node_coord_listA list of node global coordinates for the cell \(K_x\).
ky_cell_node_coord_listA list of node global coordinates for the cell \(K_y\).
sphere_centerCenter coordinates of the sphere manifold, which is stored in a row vector.
sphere_radiusThe radius of the sphere model.
pullback_kernel_valuesBEM kernel values evaluated at all area coordinates.
