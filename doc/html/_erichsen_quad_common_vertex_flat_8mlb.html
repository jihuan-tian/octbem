ErichsenQuadCommonVertexFlat - Perform the common vertex integration using the method in Erichsen1996Efficient. The integral is

\[ \int_{\hat{K} \times \hat{K}} H_{loc}(\xi, \eta) k(\xi, \eta) \intd s(\xi) \intd s(\eta), \]

.

where \(H_{loc}(\xi, \eta) = g_K(\xi) g_K(eta) \hat{\varphi}(\xi) \hat{\psi}(\eta)\) and \(k_{loc}(x, y)\) is the kernel function depending on global coordinates with \(x, y \in \mathbb{R}^3\).

Note: The integration of the product of \(H(x, y)\) and \(k(x, y)\) will be pulled back to the reference cell \(\hat{K}\) first and will further be pulled back onto the parameter space \((\omega, \eta)\) for the numerical quadrature using Erichsen's method.

Parameters
kernel_functionThe handle for the kernel function depending on global coordinates.
norder_for_etaThe number of quadrature points for \(eta\) on \(R_{d_1}\).
norder_for_omegaThe number of quadrature points for \(\omega\) on \(T_{d_2}\).
kx_basis_functionThe test function associated with a supporting point in the reference cell. Note: the set of all supporting points is used for describing the function distribution instead of the real cell geometry.
ky_basis_functionThe test function associated with a supporting point in the reference cell. Note: the set of all supporting points is used for describing the function distribution instead of the real cell geometry.

ky_shape_functions_for_geometryA list of geometric shape functions for cell \(K_y\).
kx_cell_node_coord_listA list of node global coordinates for the cell \(K_x\).
ky_cell_node_coord_listA list of node global coordinates for the cell \(K_y\).
nxThe cell normal vector for \(K_x\).
nyThe cell normal vector for \(K_y\).
JxThe surface metric (Jacobian) for \(K_x\) from reference to real cell.
JyThe surface metric (Jacobian) for \(K_y\) from reference to real cell.
retThe quadrature value.
