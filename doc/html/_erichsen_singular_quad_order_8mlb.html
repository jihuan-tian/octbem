Detailed Description

ErichsenSingularQuadOrder - Calculate the Erichsen quadrature order (actually, the number of quadrature points) for singular integrals. The integration space is \(\eta \times \omega \in R_d \times T_{4 - d}\), where \(\omega\) is firstly integreated, then \(\eta\).

Parameters
basis_function_polynomial_orderThe polynomial order \(p\) of the adopted basis function.
function_space_sobolev_orderThe Sobolev space order for the adopted function space, i.e. \(s_1\)
mesh_size_estimateAn estimate of the mesh size \(h\).
norder_for_etaNumber of quadrature points for the integration variable \(\eta\).
norder_for_omegaNumber of quadrature points for the integration variable \(\omega\).
