Detailed Description

ApplyQuadRule - Apply a quadrature rule to a function.

Parameters
functorThe function to be integrated, which should have been pulled back onto the reference cell in which the quadrature points are generated. The functor should accept a vector \(x\) as its argument, instead of a list of coordinate components.
qptsA collection of quadrature points, with dimension \(N d\), where \(N\) is the number of points, \(d\) is the space dimension.
qwtsA collection of quadrature weights, , with dimension \(N d\), where \(N\) is the number of points, \(d\) is the space dimension.
JThe Jacobian, which is a functor, maps from the reference cell to the real cell on which the functor originally is defined.
