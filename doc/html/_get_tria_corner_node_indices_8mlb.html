Detailed Description

GetTriaCornerNodeIndices - Get the indices for the corner nodes which are stored in the array of cell nodes with the specified order. Note: the ordering the triangle cell nodes depends on the implementation of the function AreaCoordsOnTriangle.

Parameters
orderThe specified order for representing the triangle's geometry.
corner_node_indicesThe indices for the corner nodes.
